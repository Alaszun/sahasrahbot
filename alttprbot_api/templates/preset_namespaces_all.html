What is a namespace?

A namespace is a way for you to share your own presets with others. You can upload your own presets to a namespace, which others then can roll games with.

By default, you can only make changes to presets in your own namespace. If you want to delegate access to namespace to others, or have a custom name for a namespace, please contact Synack for help.
